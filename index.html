「WEBコンテンツ作成の成果」

本テーマの学習目標

HTMLによってWebページの構造を記述する基本的方法が分かる,CSSによって Web ページの見た目を設定する基本的な方法がわかる,javaScriptによって Web ページのインタラクティブな処理を記述する基本的な方法がわかる状態にし、HTML, CSS, JavaScript, 画像・音声・動画ファイルを組み合わせてWebページを作成出来るようにする。

プロジェクト課題について

WWW のサーバへ検索し，サーバから届いたデータを Web ページとして整形するような Web アプリケーションを作成する。第一回ではデータ項目を参考にし、検索結果のHTMLを作った。第二回ではCSSを学び、project.html の CSS を作成した。第3回では天気予報のサンプルデータからデータの一部をコンソールに出力するプログラムを作った。第4回では前回作ったjavaScriptのプログラムを改良して，ページ上にデータを表示した。第５回では検索用の input 要素やラベル，ボタンを設置した。
プロジェクト課題で作成したページ： project.html
